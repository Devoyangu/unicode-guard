# Privacy Policy

Unicode Guard does not collect, transmit, sell, or share personal data.

The extension analyzes visible text, links, and the current tab URL locally in the user's browser to detect suspicious Unicode characters, homoglyphs, invisible characters, bidirectional controls, and unusual domain patterns.

No page content, browsing history, credentials, form data, personal communications, or personal information is sent to external servers.

User preferences are stored using Chrome extension storage.

Unicode Guard does not use remote code. All JavaScript, CSS, HTML, and other resources are packaged inside the extension.
